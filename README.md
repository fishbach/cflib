# Dependencies

* Qt >= 5.12 - https://www.qt.io/
* Botan - https://botan.randombit.net/


# Build

Qt min Version 5.12 - https://download.qt.io/archive/qt/5.12/

> qmake -r

follow Botan installation note

> make -j 4
